### 위스콘신 유방암 데이터 세트로 암 여부 판단 - 로지스틱 회귀
- 데이터프레임 만들기 : df = pd.DataFrame(cancer.data, columns=cancer.feature_names)
- 정규 분포 형태의 표준 스케일링 적용 후 train, test set 분리
- test_size=0.3
- 로지스틱 회귀를 이용해 학습 및 예측 수행
- 모형 성능 평가 
 - Confusion Matrix
 - 정확도와 roc_auc 측정
 - metrics.accuracy_score() 이용, classification_report(precision    recall  f1-score) 출력
